
# Medical Gaslighting as Marketplace Failure: A Qualitative Analysis of Women's Healthcare Experiences on Reddit

Cathryn Lara-Ramirez and Deidre Popovich

Rawls College of Business, Texas Tech University

Texas Tech Undergraduate Research Conference

# About This Project

This repository accompanies a research poster presented at the Texas Tech Undergraduate Research Conference. This study examines medical gaslighting — the dismissal or minimization of women's health concerns by healthcare providers — as a form of marketplace failure. Using qualitative discourse analysis of publicly available Reddit data, the study identifies recurring patterns in women's first-person narratives of medical dismissal and maps them onto four theoretical themes drawn from marketing and consumer research literature.

# Python Script

The script scrapes publicly available posts and comments from Reddit using a keyword-guided filtering system. It collected 197 posts and 42 comments across nine subreddits, including r/medicalgaslighting, r/WomensHealth, and r/TwoXChromosomes. A three-gate filtering system was applied to ensure posts reflected lived experiences of medical gaslighting rather than general advice-seeking. All data collected were publicly available, and this study was deemed IRB exempt.

Note: If this script is run again, it will produce new data. Reddit's feed only surfaces recent posts, so re-running the script will return different posts than those in the provided spreadsheets. The spreadsheets represent the original dataset collected for this study and should be treated as the permanent record of the data used in the analysis.



# Theoretical Framework
Findings were analyzed thematically across four key themes:

Consumer Vulnerability — Power imbalances and information asymmetry in healthcare encounters

Service Failure and Recovery — Dismissal and misdiagnosis as egregious service failures

Trust and Relationship Marketing — Erosion of provider trust and downstream effects on care-seeking

Marketplace Exclusion and Consumer Identity — Gatekeeping at intersections of gender, race, and disability


A fifth cross-cutting theme, Psychological Trauma, emerged across all four areas.
